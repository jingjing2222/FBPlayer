## 주요 기능

### 1) 축구선수 전체 검색

-   전체 선수 목록을 조회할 수 있습니다.

### 2) 이름 검색

-   선수 이름을 입력하여 해당 선수의 정보를 조회할 수 있습니다.

### 3) 포지션별 검색

-   공격수, 미드필더, 수비수, 골키퍼 등 포지션을 선택하여 선수 목록을 필터링할 수 있습니다.

### 4) 국가별 검색

-   특정 국가를 선택하여 해당 국가 선수 목록을 필터링할 수 있습니다.
